Overview

This class allows an streamed AES256 encryption of data into file using RNCryptor file format https://github.com/rnapier/RNCryptor/wiki/Data-Format

initWithURL:key:

Designated initializer with stretched, binary key. Encryption salt and HMAC salt WILL NOT BE saved in the file header https://github.com/rnapier/RNCryptor/wiki/Data-Format Warning: only use this if the key is cryptographically random and of length kCCKeySizeAES256. URL for the target encrypted file must be a file-based URL. The default PRF is kCCPRFHmacAlgSHA1. The default number of iterations is PSPDFAESDefaultPBKDFNumberOfRounds (10000).

- (id)initWithURL:(NSURL *)targetURL key:(NSData *)key
