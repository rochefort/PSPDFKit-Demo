PSPDFUndoController Class Reference

Inherits from NSObject
Declared in PSPDFUndoController.h

Overview

This is a custom undo manager that can coalesce similar changes within the same group. This class is also completely thread safe.

undoManager

Provides access to the underlying NSUndoManager. You are strongly encouraged to not use this property since it is not thread safe and PSPDFUndoController manages the state of this undo manager. However, since UIResponders can provide an undo manager, this property is exposed.
