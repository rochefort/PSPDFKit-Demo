Properties

allowFullscreen

Control if the user can enter the fullscreen mode by double-tapping. Defaults to YES.

@property (nonatomic, assign) BOOL allowFullscreen

Discussion

Note: This only affects user interaction. If you call setFullscreen:YES manually, the fullscreen mode will still be entered.
