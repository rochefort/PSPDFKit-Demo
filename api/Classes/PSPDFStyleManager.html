Inherits from NSObject
Declared in PSPDFStyleManager.h

Overview

The style manager will save UI-specific properties for annotations and apply them after creation. It also offers a selection of user-defined styles. There are three categories: Last used, key-specific and generic styles.

setLastUsedValue:forProperty:forKey:

Convenience method. Will set the last used style for key and styleProperty. value might be a boxed CGFloat, color or whatever matches the property. styleProperty is the NSString-name for the property (e.g. NSStringFromSelector(@selector(fontSize)) key is the annotation key, e.g. PSPDFAnnotationStringFreeText.

- (void)setLastUsedValue:(id)value forProperty:(NSString *)styleProperty forKey:(NSString *)key
