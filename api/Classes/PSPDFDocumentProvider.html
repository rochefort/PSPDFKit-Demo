PSPDFDocumentProvider Class Reference

Inherits from NSObject
Declared in PSPDFDocumentProvider.h

Overview

A PSPDFDocument consists of one or multiple PSPDFDocumentProvider’s. Each document provider has exactly one data source (file/data/dataProvider)

Note: This class is used within PSPDFDocument and should not be instantiated externally.
