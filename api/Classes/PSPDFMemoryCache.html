PSPDFMemoryCache Class Reference

Inherits from NSObject
Declared in PSPDFMemoryCache.h

Overview

The memory cache is designed to take up as much memory as it can possibly get (the more, the faster!) On a memory warning, it will call clearCache all release all stored images.
