PSPDFDiskCache Class Reference

Inherits from NSObject
Declared in PSPDFDiskCache.h

Overview

The disk cache is designed to store and get images including metadata in a fast way. No actual images will be held in memory (besides during the time they are scheduled for a disk write)

Properties

allowedDiskSpace

Maximum number of disk space we’re allowed to take up. In Byte. Defaults to 500MB (50010241024)

@property (nonatomic, assign) unsigned long long allowedDiskSpace

Discussion

Note: Set this to 0 to disable the disk cache.
