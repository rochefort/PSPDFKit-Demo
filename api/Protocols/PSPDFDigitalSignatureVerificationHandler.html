Class Methods

supportsFilter:withSubFilter:

Filter and subfilter are used in the signature meta data in the pdf to represent which signature handler can be used to deal with that particular signature. Example filter values are Adobe.PPKLite, Entrust.PPKEF, CICI.SignIt, and VeriSign.PPKVS. The subfilter is optional and gives further details on the encoding or format of the signature. Any handler that supports this subfilter could be used to verify the signature. Given this, an implementation might choose to ignore one of the values.

+ (BOOL)supportsFilter:(NSString *)filter withSubFilter:(NSString *)subFilter

Parameters

filter

The filter value.

subFilter

The subfilter value (can be nil).

Return Value

Whether or not this handler should be used to verify a signature with the given filter and subfilter.
