Viewing PDFs

  • PDF Viewing

    PSPDFKit supports side-scrolling, continuous scrolling and a page curl-mode. You can show one page or two pages side-by-side. You can set if the page should be fully displayed or fit to width. Even the shadows can be customized.

  • Thumbnails

    The thumbnail view can be fully customized. You can increase the thumbnail size, change or disable the filter or even completely disable thumbnail mode - for example if you want to use a scrolling bottom thumbnail bar instead.

  • Smart Zoom

    Double-Tapping on a block of text will perform a smart page content analysis and zoom exactly to that column - just as you would expect. (PDF only has the notion of single glyphs, recreating words, lines and text blocks is a challenge.)

Navigating PDFs

  • Page Links

    PSPDFKit supports page links to other pages and even other documents. There is extensive support for PDF actions, including JavaScript, named actions and even ones that control inline video, printing or email. PSPDFKit can also analyze the text and autodetect links and phone numbers.

  • PDF Outline

    PSPDFKit features a fast outline parser that deals with all the custom variants and PDF action types. Long outlines are wrapped into multiple lines, and actions can be fully customized or replaced.

  • Full-Text Search

    Our text extraction engine is one of the fastest in the industry and can deal with almost any document, language and encoding type, CMaps or UnicodeMaps. Search within documents is supported in all variants, and PSPDFKit Complete also features an sqlite-based full-text-search that can deliver near-instant results for thousands of documents.

Annotating PDFs

  • Annotating PDFs

    Add drawings, highlights, notes, stamps, free text, images, sound recordings... PSPDFKit supports all common annotation types and has a great style controller to customize them. And with global undo/redo, nothing is ever lost.

  • Annotation Inspector

    The annotation inspector features a highly customizable and intuitive color picker. With its special "half modal" mode the inspector fits perfectly on the iPhone - and on the iPad it's displayed as a popover, just as you would expect.

  • Global Copy/Paste

    Annotations can be grouped and copied to the clipboard and pasted into other documents. PSPDFKit will even convert regular text, images or links from the clipboard into the best matching annotation format when pasted.

PDF Forms

  • PDF AcroForm support

    PSPDFKit Complete/Enterprise supports most parts of the PDF AcroForms standard. You can use this to fill out forms on the go, and then submit them to a server or save the changes in the PDF itself. Web submission supports the HTTP Post format and is fully customizable.

  • Combo Box/List Box

    PSPDFKit has full support for check boxes, combo boxes and list boxes and allows adding custom values as well. Form Elements are also programmatically accessible and can be pre-filled or even dynamically loaded from a server via XFDF.

  • Form Toolbar

    The keyboard toolbar features buttons for next and previous, which are parsed as defined in the PDF spec, or simply as they are laid out in the PDF if a custom position isn't defined.

Even More

  • Embedded Videos

    PSPDFKit supports embedded video, YouTube, audio, images and web content. You can link to local or remote files, even YouTube is supported, and we go through great length to even pause the video when you go to the thumbnail view and resume as soon as you go back, and the video position will be preserved when switching pages.

  • Share via Email

    PSPDFKit ships with a customizable sharing controller for sending as email or opening in another PDF. Because not all apps can deal with PDF annotations, PSPDFKit allows "flattening", which will save the annotations readonly and in a way where they are compatible with all PDF readers.

  • Tabs

    PSPDFKit includes a controller that can deal with multiple documents and a subclass that displays documents in a tabbed interface - again fully configurable.

    If the PDF document defines PDF labels, those will be displayed instead of the regular page labels. Page labels can be roman letters, alphanumerical or custom text. Page Labels can be added with Adobe Acrobat or in-code.

    With PSPDFProcessor, you can create PDF documents on the fly from many supported file formats, such as Word (doc, docx), Excel (xls), PowerPoint, Numbers, Pages or even websites. Those documents then can be opened and annotated.

  • Accessibility

    PSPDFKit fully supports iOS accessibility and you can even let the system read the PDF content. All buttons have special accessibilityTitles so they can used even by blind people. Even small details are taken care of, like the scrobble bar which is a special touch-through area to allow navigating.

  • One document, many sources

    PSPDFKit can merge multiple document source into a single logical document, may they be from different PDF files, NSData objects or CGDocumentProviders. Everything can be mixed dynamically, for the user it will display as one file.

  • Object/Glyph Extraction

    PSPDFKit allows you to query a document for text glyphs, words, full-text and text blocks. Images can also be extract. PSPDFPageView allows easy conversion between PDF and screen coordinates.
